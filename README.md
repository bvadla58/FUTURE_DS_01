# Business Sales Performance Dashboard

## Internship Task
Future Interns – Data Science & Analytics Internship

## Task Objective
Analyze business sales data to identify:
- Revenue trends
- Regional performance
- Top-selling products
- Category-wise profit insights

## Tools Used
- Power BI
- Excel
- Superstore Dataset

## Dashboard Features
- KPI Cards
- Monthly Sales Trend
- Regional Sales Analysis
- Top 10 Products
- Profit by Category
- Interactive Filters

## Key Insights
- West region generated highest sales.
- Technology category generated maximum profit.
- September showed strong sales growth.
- Top-selling products contributed major revenue share.
- South region had lowest performance.

## Recommendations
- Increase focus on high-performing products.
- Improve marketing in low-performing regions.
- Use sales trends for inventory planning.
- Expand profitable product categories.
